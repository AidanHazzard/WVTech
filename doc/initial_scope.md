
This app is designed to help user plan their meals, manage groceries, and help stay on track with their nutrition along with time and budget goals. Users will create an account and from their add meals and recipes to a calendar. They can also choose to repeat certain meals across the days of their choosing, and automatically generate a shopping list based on these planned meals. This system can also help reduce any stress by being able to recommend recipes that fit any restictions the user may have. (i.e. allergies, budget limits, dietary restrictions, and prep time.) While keeping in consideration the ingredients that is more frequently used by the user. By combining meal planning, grocery tracking, recipes suggestions and nutrion summaries in one app, it can help solve problems such as forgetting ingredients, overspending on grociers and consistently planning meal all-together.


## Shopping List

- Allow user to add items manually to the shopping list.
- Items from upcoming meals on the calendar are added to the shopping list automatically.
- Use a grocery shopping API to allow the user to quickly purchase items from a local store for pick-up or delivery.
- Remove items from shopping list that are in pantry automatically


## Pantry

- The user can use a UPC bar code scanner to add items to their pantry
- Allow user to add items manually to the pantry
- The system can remove items used in meals automatically
- The system can remove perishable items from pantry automatically that have gone beyond their use date
- The system can remind the user to use perishable items in the pantry at the users request.
- The user can add leftovers from meals to the pantry. The recommendation system may then use the leftovers for future meals.


## Recipe Book

- Allow users to rate recipes on a 1-5 scale
- Aggregate user ratings and allow users to search/sort recipes by rating
- Users can Favorite/Star recipes to quickly search for recipes they like
- Allow user to add their own recipes to their recipe book
- The user can search for recipes to add to meals. Search results are ordered by chosen categories of recipes and filtered by the user's dietary requirements.
- The system uses a grocery shopping API to approximate cost local to the user


## Budget

- The user can set a per meal/day/week/month budget
- Allow the user to set a tolerance of how much they value staying within their given budget.


## Time Management

- The user can set the amount of time they would like to spend on each meal.
- Allow the user to set a tolerance of how much they value staying within their time budget


## Diet

- The system can track user nutrient consumption from meals and report back to the user their nutrient consumption based on a provided time period.
- The user can set and manage allergies and other dietary restrictions.
- The user can set their nutritional consumption requirements
- The user can set a preference for what times they would like an amount of nutrients.
- The user can set a tolerance of how closely they would like to follow nutritional consumption requirements
- Allow user to select a texture modified diet if necessary.
  

## Meal Recommendation

- Recommend meals to user based on their budget, time budget, diet, pantry, and preferences inferred from favorited recipes and recipe ratings they have given.
- Allow the user to balance how much they value staying within budget and their preferred meals
- Allow the user to select certain categories of food they would like for a given meal. For example: breakfast, lunch, dinner, italian, mediterranean, tex-mex, etc.
- The system can combine multiple recipes to make a single meal to meet nutritional requirements


## User Account

- A user can create and sign-in to an account
  

## Calendar

- Allow the user to set the times and dates for meals
- Allow the user to repeat meals on a regular basis
- Allow the user to set days and times for shopping and manage the shopping list accordingly
- When adding a meal to the calendar, the system can recommend a meal to add
- The user can set days to prepare meals. The system will collate necessary prep work needed for meals in between prep days from recipes and provide a list of instructions to the user.
- The system can send notifications to the user to remind them when to start preparing and cooking a meal. The user can select which notifications they would like to recieve.
